This tool analyzes a software and shows all used C++ language features according to C++ standards. Just copy into the root directory and run in cmd. 

1. Clone the repo
2. Open the Visual Studio solution file CppProgAnalyzer.sln
3. Build the Run configuration
4. Copy the CppProgAnalyzer.exe into the root directory of Your project
5. Run the CppProgAnalyzer.exe from PowerShell or cmd
